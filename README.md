## Deksripsi
Aplikasi ini merupakan apk login yang mengimpelmentasikan autentikasi berupa login memakai email, biometric, password reset, email verification dan login dengan role berbeda.

## Fitur apk
* terdapat sistem keamanan & autentikasi
* login
* register
* reset passowrd
* login dengan role berbeda
* biometric

## Dependencies
* @react-native-async-storage/async-storage

* @react-navigation/native-stack
* expo
* expo-local-authentication
* expo-secure-store
* firebase
* react
* react-native
* react-native-safe-area-context
* react-native-screens

## Tugas Mandiri
Role-based authorization (admin vs user) dengan Firestore + custom claim.
